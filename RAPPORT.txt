Vous devrez créer un site de A à Z, « à la main », en utilisant les contenus suivants :
– HTML5 / CSS3 ;
– Javascript, jQuery (et ses plugins), AJAX ;
– le format JSON ;
– au moins une API de votre choix ;
– d’autres technologies côté client de votre choix (facultatif).
En outre, le site devra avoir un design web « responsive » (lisible à la fois sur un ordinateur, un
smartphone, une tablette, etc.).
